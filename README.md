# Study Planner

A simple web-based study planner that helps users organize and manage study tasks. Users can add tasks, assign priority levels, and delete completed tasks through an easy-to-use interface.

## Live Demo
https://mohammedali-fatteh.github.io/Study-Planner/

## Features
- Add study tasks
- Set task priority (High, Medium, Low)
- Delete tasks when completed
- Simple and clean user interface
- Color-coded priority levels

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run Locally

Clone the repository:

```bash
git clone https://github.com/mohammedali-fatteh/Study-Planner.git
```

Open the project folder and run:

```
index.html
```

in your browser.
